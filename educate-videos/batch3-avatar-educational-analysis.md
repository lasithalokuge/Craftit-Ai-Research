## Video 5: "The AI Video Workflow BLOWING UP Businesses in 2026" (Wes McDowell)
**URL:** https://www.youtube.com/watch?v=bzoTOgqD15I
**Creator:** Wes McDowell
**Format:** Strategy/workflow guide for business YouTube (~18 min)

### Five AI Workflows Covered

#### Workflow 1: Topic & Title Research
- **Tool:** ChatGPT Agent Mode (not regular deep research)
- Agent mode can open YouTube, type into search bar, gather real search data
- Prompt specifies: find topics for niche authority + real keyword phrases + compelling titles
- Output: two columns — Column A (keyword-based topic) + Column B (finished title with hook)
- Can generate entire year's content plan

#### Workflow 2: Thumbnail Creation
- **Google Nano Banana** (AI image generator) — inconsistent quality, wrong dimensions
- **Pixels** (recommended) — upload 20 photos to train on your face, then:
  - Text prompt → thumbnail
  - Handles text well, knows thumbnail dimensions
  - **Recreate feature** — paste existing YouTube thumbnail URL → remake with your face/branding
- Concept: "ethically stealing" proven thumbnail layouts (endorsed by Patty Galloway)

#### Workflow 3: Research
- **Tool:** ChatGPT Deep Research mode
- Input: title + video description + brain dump of known content
- Asks for: stats, case studies, additional tips, arguments
- Key instruction: "don't take irrelevant side quests or tangents"
- Takes ~15 minutes to research
- Output: structured research document for scripting

#### Workflow 4: Script Writing
- **Tool:** ChatGPT Custom GPT or Claude Project (persistent scripting agent)
- Training materials needed:
  - Script formula document (video structure template)
  - Writing sample (transcript of natural speech OR aspirational YouTuber's transcript ~10 min)
- Creates reusable agent — upload training materials once, use forever
- Output: 85% complete first draft (add 15% personal touch)
- Claude noted as "still a little better at writing like a human"

#### Workflow 5: B-Roll Generation
- **Tool:** Sora 2 (Cameo feature)
- Train AI on your face (record short clip saying 3 numbers + head turn)
- Generate custom B-roll footage featuring virtual you (working with clients, speaking on stage, etc.)
- Also: motion graphics generation from text prompts
- Key distinction: enhancing already-good videos, NOT replacing human element
- Example: "futuristic purple and black gradient background with four glowing icons" → generated in minutes vs hours of editor work

### Key Insights
- AI handles ~80% of video production work
- The 5-workflow pipeline: Topics/Titles → Thumbnails → Research → Script → B-Roll
- **Custom GPT/Claude Project as reusable scripting agent** is a powerful pattern
- Sora 2 Cameo for personalized B-roll is the differentiator
- Total time: blank screen to finished video in 2-3 hours (vs days manually)
- NOT about fully AI-generated videos — about AI-augmented human content

### Craftit Relevance
While not avatar-specific, this shows the **complete content creation pipeline** that Craftit's EDUCATE category needs to support. The custom scripting agent pattern and research workflow are directly applicable.

### What Craftit Can Learn: Automation Opportunities

#### 1. Document-to-Video Pipeline (Highest Impact)
Existing training materials (PDFs, PPTs, manuals) → avatar-presented videos. This is the #1 feature HeyGen promotes for training. Craftit should replicate this: upload document → auto-extract content → generate script → produce video.

#### 2. Prompt-to-Structured-Plan Pattern
User describes training goal + audience → AI generates complete video plan (scenes, narration, visuals, timing). User reviews for clarity, not technical editing. This "review like a creator" UX is the target mental model.

#### 3. Brand Kit with Auto-Detection
Paste company URL → auto-extract logo, colors, fonts. This removes a massive friction point for corporate users. Brand Voice settings preserve terminology across translations.

#### 4. Script Generation from Training Goal
Input: "Training goal: teach new employees our CRM system. Audience: non-technical staff." → complete structured script. This goal+audience → script pattern is simpler than asking users to write.

#### 5. Avatar Consistency as Trust Builder
Same avatar across all training content builds viewer trust and brand recognition. Craftit should default to consistent avatar selection, not random variety.

#### 6. Quality Control Checklist (Automated)
- Lip sync naturalness
- Pacing (not rushed/robotic)
- Text readability
- Movement-tone alignment
Craftit can automate these checks before delivery.

#### 7. Multi-Format Output
Single source → multiple formats (1080p, 720p, square for social, with/without captions). Automated format variants from one production.

### What Makes Avatar Educational Videos Effective vs Robotic/Boring

#### DO (Effective):
1. **Match avatar to audience** — professional for corporate, casual for social
2. **Use consistent avatar** — builds trust through familiarity
3. **Add visual variety** — text overlays, images, highlights at key script moments
4. **Include branded elements** — logo, colors, intro/outro (looks custom, not generic AI)
5. **Keep natural pacing** — not too fast, not robotic
6. **Post-produce** — branded intro/outro, transitions, balanced audio
7. **Focus on message clarity** — avatars work because viewers focus on content, not presenter
8. **Use appropriate gestures/animations** — subtle movement prevents "uncanny valley"
9. **Add captions** — accessibility + engagement boost
10. **Leverage translation** — same quality across all languages

#### DON'T (Robotic/Boring):
1. Don't deliver raw AI output without any polish
2. Don't use mismatched avatar (casual avatar for compliance training)
3. Don't skip quality control (bad lip sync kills credibility)
4. Don't use generic backgrounds (match setting to topic)
5. Don't make scripts too long without visual variety
6. Don't ignore pacing — monotone delivery is the #1 complaint
7. Don't forget CTA and clear structure

### Key Takeaways for Craftit EDUCATE Category

1. **The killer input is existing documents** — PDF/PPT → video is the highest-value automation for corporate training
2. **Prompt-driven creation** beats template-driven — "describe your training goal" > "pick a template"
3. **Review-not-edit UX** — users should review a plan for clarity, not manage timelines
4. **Brand consistency automation** — URL → brand kit is table stakes
5. **Translation is a massive multiplier** — create once, deploy globally
6. **Avatar consistency > avatar variety** — recommend sticking with one avatar per brand/channel
7. **Post-production polish** differentiates quality — Craftit should automate intro/outro, text overlays, captions
8. **LMS integration** is expected for corporate training distribution
9. **Screen recording integration** needed for software training use case
10. **The "85% draft" model works** — AI generates near-complete video, user adds final 15% touches
